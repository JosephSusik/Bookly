\setabstractcz{
Tato diplomová práce je zaměřena na vývoj aplikace, která lidem pomůže spravovat jejich osobní knihovnu v digitální podobě. 
Uživatelé si mohou přidávat knihy jednoduše naskenováním ISBN kódu nebo ručním zadáním, prohlížet si informace o knihách a sledovat, co~už přečetli nebo chtějí číst. 
Důležitou součástí aplikace je systém doporučování knih, který uživatelům nabízí tipy na další tituly, které by je mohly zajímat, a to na základě jejich čtenářských preferencí a historie. 
Doporučení jsou vytvářena pomocí strojového učení. 
Aplikace je dostupná jak ve webové, tak v mobilní verzi. Webová aplikace je implementována ve~frameworku Next.js s využitím jazyka TypeScript. Mobilní aplikace je implementována v~open-source frameworku Flutter.
}

\setabstracten{
This master's thesis focuses on the development of an apllication designed to help users manage their personal book library in digital form.
Users can add books either by scanning the ISBN code or by entering the data manually, browse detailed information about individual books, and track which titles they have already read or plan to read.
An important part of~the application is a book recommendation system that suggests additional titles based on~users' reading preferences and history. The recommendations are generated using machine learning algorithms.
The application is available in both web and mobile versions. The web application is implemented using the Next.js framework with TypeScript. The mobile application is implemented using the Flutter framework.
}
